## 📖 About

A Scientific Calculator developed as a **Source Code Management** project.

The calculator is available in two versions:

* 🐍 **Python (CLI Version)** – Runs directly in the terminal.
* 🌐 **Web Version** – Built using HTML, CSS and JavaScript.

The goal of this project is to demonstrate collaborative software development using Git and GitHub while implementing common scientific calculations.

# 🚀 Getting Started

## Python Version

Clone the repository

```bash
git clone https://github.com/waveboard25/scientific_calc-.git
```

Go into the project folder

```bash
cd scientific_calc-
```

Run

```bash
python main.py
```

---

## Website Version

Simply open

```
index.html
```

in your browser.

No installation required.
